SignalCatch

Bybit 우선 API 대기 마지막 갱신: 없음

시장 데이터를 새로고침하면 통합 방향성이 표시됩니다.

사용 방법
  1. 스캐너에서 Bybit USDT 무기한 선물 상위 종목을 스캔합니다. 기본값은 100개이며 10개부터 200개까지 조정할 수 있습니다.
  2. 종목의 현재 셋업을 열어 모든 타임프레임을 확인한 뒤 판단합니다.
  3. 진입 구간, SL, TP는 현재가 기준 분석값입니다. Daily와 Swing 셋업은 분할 진입과 분할 TP 안내도 함께 표시합니다.
  4. 백테스트는 보조 확인용으로만 사용합니다. 과거 승률이 다음 거래의 수익을 보장하지 않습니다.
  5. SignalCatch는 분석 대시보드입니다. 주문을 실행하지 않으며 투자 조언이 아닙니다.
